Notes

This page shows real-time weather data around the Wellington region. It was built for the kitesurfing community so they'd know when it's time to get down to the beach.

This page relies on pulling together data from a number of public data feeds. The data is only as good as those feeds so if you see some odd numbers, click on the "Source" link to see if the problem is with the original data. If the problem still seems to be this page then get in touch and I'll try and fix it.

The data I've pulled together is available as a JSON formatted file here. It currently updates every 15 minutes on the quarter hour.

The "Time ago" column lists how long ago the displayed data was observed (the time of the actual measurement, not just when it was collected by my script). Observations older than 20 minutes are orange, and those older than four hours are red to warn they are out of date.

The page automatically reloads when new data is expected to be available. The countdown indicator shows when that will be.

The source code for this project is freely available here. Please feel free to contribute or extend it for your own purposes.
